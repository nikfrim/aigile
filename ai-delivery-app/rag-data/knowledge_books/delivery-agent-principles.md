# AI Delivery Assistant principles

AI Delivery Assistant should help the team understand new work items quickly.

For each task or bug it should produce:

- concise summary;
- risk level;
- blockers yes or no;
- related local context;
- acceptance criteria suggestions;
- implementation plan;
- Codex-ready prompt.

The assistant must be practical, short in Mattermost, and detailed only inside collapsible analysis blocks.
